# Script Editor Interface - User Guide

## Overview

The Script Editor Interface is the heart of FanCast AI's professional content creation workflow. This powerful tool transforms your story concepts into editable, professional-grade scripts with advanced TTS (Text-to-Speech) annotation capabilities and character voice management.

## Getting Started

### Accessing the Script Editor

1. **Navigate to Creator Studio** - Click the "Create" tab in the bottom navigation
2. **Start Creating** - Click "Start Creating" to begin a new story
3. **Enter Your Story Concept** - Describe your story in the text area
4. **Configure Storytelling Options** - Expand and customize your preferences
5. **Generate Series** - Click the "Generate Series" button
6. **AI Processing** - Wait for the "AI Crafting Your Script..." loading screen
7. **Script Editor Opens** - The professional editing interface appears

### Interface Layout

The Script Editor features a clean, professional layout optimized for mobile and desktop:

- **Header Bar**: Series title, episode information, and action buttons
- **Side Panel**: Characters & Voices management
- **Main Editor**: Line-by-line script editing area
- **Bottom Actions**: Cost calculation and generation controls

## Workflow Guide

### Complete Creation Process

1. **Story Concept Entry**
   - Enter detailed story description
   - Include character information and setting details
   - Rich details create better AI-generated scripts

2. **AI Script Generation**
   - Professional loading animation with progress indicators
   - Typically takes 3-5 seconds for script generation
   - Automatic transition to Script Editor Interface

3. **Script Refinement**
   - Review AI-generated script lines
   - Edit text content for perfect dialogue and narration
   - Assign appropriate characters to dialogue lines
   - Add TTS annotations for emotional depth

4. **Character Voice Management**
   - Verify character voice assignments
   - Use preview buttons to test voice quality
   - Ensure consistent character voices throughout

5. **Final Review & Generation**
   - Review complete script for quality and consistency
   - Verify cost calculation and duration estimate
   - Click "Generate Audio" for final production

### Best Practices

#### Script Editing
- **Clear Character Distinction**: Ensure dialogue is clearly attributed
- **Natural Language**: Edit for natural speech patterns
- **Emotional Depth**: Use TTS annotations strategically for impact
- **Consistent Tone**: Maintain character voice consistency

#### TTS Annotations
- **Strategic Usage**: Don't over-annotate; use for key emotional moments
- **Character Appropriate**: Match annotations to character personality
- **Scene Context**: Consider the scene's emotional context
- **Preview Testing**: Use preview buttons to verify annotation effects

#### Character Management
- **Voice Consistency**: Keep the same voice for each character throughout
- **Clear Attribution**: Ensure every dialogue line has proper speaker assignment
- **Narrator Usage**: Use narrator voice for scene descriptions and transitions

## Troubleshooting

### Common Issues

#### Script Not Loading
- Ensure story concept was entered before clicking "Generate Series"
- Wait for loading animation to complete
- Refresh page if loading stalls

#### Annotations Not Appearing
- Click directly on the üé≠ button for each line
- Select annotation from the modal popup
- Annotation tags should appear immediately

#### Character Assignment Issues
- Use dropdown menus to assign speakers to dialogue lines
- Narrator is automatically available for narration lines
- Ensure consistent character names throughout

#### Preview Not Working
- Preview buttons (‚ñ∂Ô∏è) simulate audio playback
- Actual audio generation requires backend integration
- Use for interface testing and workflow validation

### Performance Tips

- **Efficient Editing**: Make major edits before adding annotations
- **Strategic Previews**: Use preview buttons selectively to avoid overload
- **Save Regularly**: Use the save button to preserve progress
- **Mobile Performance**: Close other apps for optimal mobile performance
